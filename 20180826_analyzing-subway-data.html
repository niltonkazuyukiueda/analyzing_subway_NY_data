Subway Data Analysis¶

Introdução¶

O sistema de nibus e trens de Nova Iorque - o Metro Transit Authority - fornece seus dados para download atravs de arquivos csv. Uma das informaes disponveis so os dados das catracas do metr que contm logs semanais de entradas cumulativas e sadas por catraca por estao de metr em algum intervalo de tempo.

Neste projeto iremos utilizar apenas os das catraca disponveis em: http://web.mta.info/developers/turnstile.html.

Sobre este projeto¶

Neste projeto voc ir aplicar todos os conhecimentos adquiridos neste primeiro ms de curso. Iremos praticar tarefas bsicas de aquisio, limpeza de dados e nesse processo iremos descobrir coisas essenciais sobre os dados utilizando o que foi aprendido no curso de estatstica.

O objetivo deste projeto explorar a relao entre os dados das catracas do metro de Nova Iorque e o clima no dia da coleta. Para isso, alm dos dados do metr, precisaremos os dados de clima da cidade de Nova Iorque.

Os principais pontos que sero verificados neste trabalho:

  • Coleta de dados da internet
  • Utilizao de estatstica para anlise de dados
  • Manipulao de dados e criao de grficos simples com o Pandas

def create_master_turnstile_file(filenames, output_file):
    with open(output_file, 'w') as master_file:
        master_file.write('C/A,UNIT,SCP,STATION, LINENAME, DIVISION, DATEn,TIMEn,DESCn,ENTRIESn,EXITSn\n')
        for filename in filenames:
            with open(filename) as partial_file:
                for line in partial_file:
                    master_file.write(line)

import sys

def mapper():

    for line in sys.stdin:
        data = line.strip().split(",")
        
        if len(data) >= 7:
            unit = data[1]
            entries = data[6]
        
            print(f"{unit}\t{entries}")

def reducer():
    entriesTotal = 0
    oldKey = None

    for line in sys.stdin:
        data_mapped = line.strip().split("\t")
        if len(data_mapped) != 2:
            continue
        
        thisKey, thisEntry = data_mapped
        
        try:
            thisEntry_to_f = float(thisEntry)
            if oldKey and oldKey != thisKey:
                print(f"{oldKey}\t{entriesTotal}")
                oldKey = thisKey
                entriesTotal = 0
            
            oldKey = thisKey
            entriesTotal += thisEntry_to_f
        except ValueError:
            pass
            
sys.stdin = open('mapper_result.txt')
sys.stdout = open('reducer_result.txt', 'w')
reducer()
